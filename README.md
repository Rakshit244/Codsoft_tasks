# Python Programming Internship

Welcome to my **CodSoft Python Programming Internship** repository. This repository contains the projects I completed during my internship, demonstrating my understanding of Python programming fundamentals, problem-solving, and application development.

## Internship Tasks

### 🧮 Task 1: Simple Calculator

A Python-based calculator that performs basic arithmetic operations such as:

* Addition
* Subtraction
* Multiplication
* Division

The project includes:

* **Command-Line Interface (CLI)** version
* **Graphical User Interface (GUI)** built using **Tkinter**
* Input validation and error handling for a better user experience.

---

### 🔐 Task 2: Password Generator

A secure password generator that creates random passwords based on user preferences.

**Features:**

* Custom password length
* Uppercase letters
* Lowercase letters
* Numbers
* Special characters
* Input validation
* Multiple password generation in one session

The project uses Python's built-in **random** and **string** modules to generate strong and secure passwords.

---

### 📒 Task 3: Contact Book

A command-line Contact Book application that allows users to manage their contacts efficiently.

**Features:**

* Add new contacts
* View all contacts
* Search contacts by name
* Update contact details
* Delete contacts
* Store name, phone number, email, and address

This project demonstrates the implementation of basic **CRUD (Create, Read, Update, Delete)** operations using Python.
