# Contact Book - CodSoft Internship Project

## Project Overview

This project is a **Contact Book** application developed using Python during my **CodSoft Python Programming Internship**. It is a command-line application that allows users to manage contact information efficiently by adding, viewing, searching, updating, and deleting contacts.

The project demonstrates the use of Python lists, functions, loops, and conditional statements to implement a simple contact management system.

---

## Features

*  Add new contacts
*  View all saved contacts
*  Search contacts by name
*  Update existing contact details
*  Delete contacts
*  Store name, phone number, email, and address
*  Menu-driven command-line interface

---

## Technologies Used

* Python 3

---

## 🚀 How to Run

Run the application using:

```bash
python ContactBook.py
```

The program will display a menu with the following options:

```text
1. Add Contact
2. View Contact
3. Search Contact
4. Update Contact
5. Delete Contact
6. Exit
```

Enter the corresponding number to perform the desired operation.

---

## Features Explained

### Add Contact

Allows users to save a new contact by entering:

* Name
* Phone Number
* Email Address
* Address

### View Contacts

Displays all contacts currently stored in the contact book.

### Search Contact

Searches for a contact using the contact's name and displays the stored information if found.

### Update Contact

Finds an existing contact by name and allows the user to modify:

* Name
* Phone Number
* Email
* Address

### Delete Contact

Removes a contact from the contact book using the contact's name.
